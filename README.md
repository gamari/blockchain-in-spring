# 概要

Javaでブロックチェーンを実装してみる。


# 機能

- transaction機能
- mining機能
- wallet機能


# 注意

- Sync機能は未実装。
- P2P機能はなし。
	- RestControllerで代用。
- マイニング処理は擬似的にフロント側が叩いて行っています。
- Addressの永続化はしてません
	- そのため、Addressの存在有無判定はしていません。
